1. youtube-dl
  - biblioteca em python
  - pode executar atravês da consola depois de instalada
  - Formatos de video:
    * mp4, 3gp, flv, webm
  - Formatos de audio:
    * m4a, webm

2. PAFY
  - biblioteca em python
  - poucos formatos de video comparativamente com o youtube-dl
  - Formatos de video:
    * mp4, 3gp, flv, webm
